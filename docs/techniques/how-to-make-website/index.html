如何在半天内搭建一个博客以及上线？

这里分享一个如何在半天内搭建一个博客以及上线的方法，也就是整个网站的制作过程。

首先，为什么选择自己建一个网站，而不用石墨、飞书这些资源？

最大的原因是免费！！！

我尝试使用石墨、飞书这些网站，结果发现多人协同都在一定程度上是收费的，而石墨这种单纯的文档整理，好像并不符合我想要快速信息检索的需求。所以我把目光投向开源框架，这时候有一个英雄给我推荐了一个教程网站，我发现这个教程是用docusaurus前端框架+部署在github上完成的，全程不需要花一分钱。而且这个框架依托FaceBook和GitHub两个大平台，不用考虑不能用的问题。

制作细节​

下面详细介绍下具体是如何制作的吧。

环境设置​

你需要安装以下环境：

node.js
git

创建网站​

你需要在你所在的文件位置打开终端，然后输入：

npx create-docusaurus@latest my-website classic

其中my-website是指你想创建的文件夹的名字

打开网站​

进入你的网站所在的文件夹：

cd my-website

打开网站：

npm run start

这时候，你的浏览器将会自动打开http://localhost:3000

点击这个链接进入，你就可以看到自己的网站已经建完啦！

部署网站​

那么是不是到这一步就结束了呢？

不是的，当前的网站只有在你打开着电脑且只有你这台电脑能够访问，也就是只能本地访问（因为域名是localhost）。如果你想让自己的网站给更多人看到，那就要进行网站部署。

这里我使用的方式是用github部署。GitHub是一个知名的开源仓库平台，你可以在这里存储、下载你需要的文件，也可以进行工作协同。

第一步就是创建仓库。

新建分支​

这个框架在文档中说，默认使用的分支是gh-pages，所以需要你创建一个新的分支。

设置部署​

修改配置​

进入你的网站文件的docusaurus.config.js文件

按照注释修改github相应配置

打包部署​

打开终端，在你的网站文件路径下输入命令

cmd /C "set "GIT_USER=<GITHUB_USERNAME>" && yarn deploy"

你的网站就上线啦！
